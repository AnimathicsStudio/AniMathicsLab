# Streamlit Labs

Source app Python/Streamlit untuk eksperimen komputasional AniMathics Lab.

Folder ini dipakai untuk development dan deploy ke Streamlit/GitHub. Katalog web utama tetap membaca metadata dari `../data/eksperimen.JSON`.

Konvensi tiap lab:

```text
nama-lab/
├─ app.py
├─ *_core.py atau *_engine.py
├─ *_plot.py
├─ requirements.txt
└─ sample_data/ atau file contoh bila perlu
```
