# Fuzzy Forecasting Lab

Applet Streamlit untuk peramalan runtun waktu berbasis fuzzy.

Metode:

- Fuzzy Time Series
- Fuzzy Markov Chain

Fitur:

- Dataset contoh penjualan UMKM
- Editor data
- Upload CSV opsional
- Interval fuzzy
- Fuzzifikasi data
- Fuzzy logical relationship
- Prediksi periode berikutnya
- Grafik aktual vs prediksi
- Evaluasi MAE, MAPE, RMSE

Jalankan lokal:

```bash
streamlit run app.py
```
