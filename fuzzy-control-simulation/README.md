# Fuzzy Control and Simulation Lab

Applet Streamlit untuk simulasi sistem kontrol fuzzy.

Mode kasus:

- Kipas Otomatis
- Penyiraman Tanaman Otomatis

Metode:

- Mamdani
- Sugeno

Fitur:

- Slider input sensor
- Fungsi keanggotaan input
- Rule switch
- Tabel aturan aktif
- Respons sistem
- Agregasi Mamdani
- Rata-rata berbobot Sugeno
- Surface plot input-output

Jalankan lokal:

```bash
streamlit run app.py
```
